Internet como surgiu

Na década de 1950 - Guerra Fria tivemos a corrida espacial onde se obteve um grande crescimento na ceara tecnológica, como o surgimento da DARPA(Defense Advanced Rearch Projects Agency)

Em 1958 se tratarmos a internet como comunicação, teremos telégrafo com morse code

Arpanet primeira comunicação em pacotes, enviados um a um em 29/10/1969 California à Stanford, eviando a palavra "login", mas chegou somente "lo", ou seja neste periodo tivemos a primeira comunicação e a primeira queda

Grandes nomes envolvidos

Grace Murray Hopper - uma das primeiras programadoras do Mark1, criadora da linguagem Flow-Matic.

Robert E. Kahn - primeira apresentação da ARPANet e primeiro e-mail
Criador do TCP/IP

Joseph Carl - teorizou sobre uma rede onde todos poderiam se comunicar, plantando a semente da comunicação

Marc Andreessen - (re)inventou o navegador o Netscape feito a partir do Mosaic, tornando-o mais amigavel com recursos

Linguagem de programação(Server-side e Cliente-side)

Server-side - são linguagens que o servidor ira processar e mandar para seu navegador, estas são:

  • PHP
  • ASP
  • Java
  • Ruby
  • Python

Cliente-side - são lingaguens para o navegador entender, assim o processamento sera feito pelo browser, sendo estas:

  • JavaScript

Linguagem de marcação HTML

O HTML é uma linguagem utilizada para criar páginas web por meio de marcadores (tags) e atributos, que definiram como o conteúdo sera apresentado em um navegador web
